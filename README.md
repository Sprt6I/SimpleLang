Simple programing language line Basic, written in c++. Each instruction must be written in new line

Commands:
* print  -> prints something to the screen
  print something_to_print
* int    -> declares int variable
  int variable_name value
* string -> declares string variable
  string variable_name "value"
* add    -> add something to the variable
  add variable_name value
* sub    -> substracts something from the variable
  sub variable_name value
* mul    -> multiplies variable by something
  mul variable_name value
* div    -> divides variable by something            // Not Implemented
* for    -> basc for loop
  for variable_name start stop end
  // what ur loop does
  end
* def    -> creates function                         // Not Implemented
